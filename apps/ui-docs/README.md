# @hbdlzy/ui-docs

`@hbdlzy/ui-docs` 是给人看的文档入口。

当前包含：

- workspace 包总览
- AI manifest 路径说明
- `BaseCard` 可运行示例
- `BaseEChart` 可运行示例
- `BaseExportButton` 可运行示例
- `BaseTable` 可运行示例
- `OutlinedInput` 可运行示例
- `OutlinedSelect` 可运行示例
- `OutlinedDatePicker` 可运行示例
- `OutlinedDateTimePicker` 可运行示例
- `OutlinedTimePicker` 可运行示例
- `OutlinedCascader` 可运行示例
- `OutlinedTreeSelect` 可运行示例

启动方式：

```bash
pnpm --filter @hbdlzy/ui-docs dev
```

构建方式：

```bash
pnpm --filter @hbdlzy/ui-docs build
```
